# Instruções de Uso

## Requisitos
- Python 3.x
- Bibliotecas: tkinter, PIL (Pillow), sqlite3

## Instalação
1. Certifique-se de ter o Python 3.x instalado.
2. Instale a biblioteca Pillow com o comando: pip install pillow.

## Como Usar
1. Execute o script Python.
2. Preencha os campos de Nome, CPF e Estado.
3. Clique no botão "Salvar" para armazenar os dados no banco de dados SQLite.
4. A imagem de fundo será carregada automaticamente a partir do arquivo 'ceuazul.png' localizado no mesmo diretório do script.
